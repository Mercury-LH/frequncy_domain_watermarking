# Product

## Register

brand

## Users

点开分享链接的访客：同学、老师、招聘者，以及想给自己图片加隐形水印的普通用户。他们大多第一次接触"频域水印"概念，停留时间短，用手机或电脑浏览器直接访问，无需注册。核心任务链：看懂这是什么 → 上传图片和自定义水印（文字或图片）→ 得到含隐形水印的图 → 提取验证 → （可选）攻击实验看鲁棒性。

## Product Purpose

把"图像频域水印"研究项目（DFT / DCT / DWT 三种算法，含攻击与指标体系）转化为一个免费、开箱即用的在线体验。成功的样子：访客点开链接的第一屏就被打动，三分钟内完成一次"嵌入 → 提取"闭环，并直观感受到"水印藏在频率里、肉眼看不见但提取得回来"的魔力。展示优先，工具其次——这是一件作品，不只是一个实用程序。

## Brand Personality

学术、克制、可信。像一篇排版精致的论文变成了网站：明亮纸感底色、衬线标题、精确的数据呈现、冷静而流畅的动效。整体质感对标北美一线科技公司的官网水准（Stripe 式白底克制 × 学术出版物的严谨气质）。

## Anti-references

- 暗色"黑客风"仪表盘、霓虹渐变科技感——已被用户明确否掉。
- 玻璃拟态 + 紫粉渐变的大众 SaaS 模板脸。
- 奶油色/米黄暖白底的 "AI 默认纸感"——纸感由排版、衬线与留白传达，底色用零彩度或偏品牌绿色相的真正灰白，不用暖米色。
- 杂志式 editorial 装腔（意大利体大标题 + 小号 mono 标签 + 通栏细线的模板化组合）。
- 每个区块上方重复的小号全大写 eyebrow 标签、01/02/03 编号脚手架。

## Design Principles

1. **把看不见的展示出来**：频谱图、水印差值图、提取对比是最好的视觉素材——用真实的信号处理产物当"插画"，不用装饰性图形。
2. **论文的严谨，产品的顺滑**：指标（PSNR/SSIM/NC/BER）像论文表格一样精确呈现，交互像大厂产品一样零摩擦。
3. **三分钟魔法闭环**：任何访客不看说明书就能完成嵌入 → 提取，每一步都有即时视觉反馈。
4. **展示领路，工具随行**：首屏讲故事、造氛围；工具区块自然融入叙事，而不是把访客直接扔进控制台。
5. **克制的动效**：动效只用于揭示状态与引导视线，150–250ms，出场即消失；全站适配 prefers-reduced-motion。

## Accessibility & Inclusion

WCAG AA：正文对比度 ≥ 4.5:1，大字号 ≥ 3:1；全键盘可操作；语义化 HTML 与 ARIA 标签；prefers-reduced-motion 提供交叉淡化替代；中英双语切换（默认中文）。
